emacs-for-python by gabrielelanaro

Collection of emacs extensions specifically collected for python development, with workflow guidelines!

The extensions collection is designed to be extremely pluggable in your emacs environment. After you've plugged in this package you have a perfect environment for python development and you can start immediately to use it. It powers a lot of features, that comes from very notorious packages so the extensions used are extremely stable, some of them are:

  • ido prompts
  • autocompletion + rope (if you have pymacs)
  • snippets, simplified and standard-compliant
  • automatic error/warning highlighting with flymake (powered by pyflakes)
  • custom keybindings
  • much more (really)
For a more comprehensive list look at the README.

Dependencies

  • emacs23
  • pyflakes
  • pymacs (optional but extremely recommended)

Install

unpack the archive and add 

(load-file "/path/to/epy-init.el")
to your .emacs
